NACA_Project
------------

Application developed by Mattia Tristo (Academic Degree in Mechanical Engineering).

It writes scripts for Autodesk/Autocad and Dassault/Draftsight starting from 4 digits containing the mean line, upper and lower border and closing array.

To import the script open your CAD and write at the command line SCRIPT, it opens a dialog, choose the file and press ok. The drawing will appear.

The distro file contains applications for Mac OS X, Linux, Windows.
